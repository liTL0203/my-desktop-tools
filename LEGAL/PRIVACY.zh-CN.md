本隐私政策描述了「桌面工具箱」（以下简称"本应用"）如何收集、使用和保护您的数据。

## 数据收集分级

我们采用三级数据收集模型：

### Level 0 — 核心运行数据（可选，Opt-in）

启用后，将收集以下数据点：

- 应用启动/退出事件（含会话时长，单位秒）
- **会话初始化事件**（应用启动时发送一次 pageview，用于在分析仪表盘中建立访客会话；驱动访客总数、国家分布、来源统计等聚合数据）
- 应用版本号
- 操作系统类型（Windows/macOS/Linux）
- CPU 架构（如 x86_64、aarch64）
- 界面语言（如 zh-CN、en-US）
- 匿名设备标识（随机生成的 UUID v4，不基于硬件指纹）
- 屏幕分辨率（如 1920x1080，仅用于设备类型统计）

此外，每次分析请求会携带以下**固定元数据**（所有用户共享的常量值，不含任何个人或设备特定信息）：

- `url`：虚拟路由标识（如 `app://main`、`app://plugin/pomodoro`）
- `title`：固定应用名称「My Desktop Tools」
- `hostname`：固定伪主机名 `app.my-desktop-tools.local`
- `referrer`：固定项目主页 URL（开源仓库链接）

这些固定值仅用于分析仪表盘的路由/上下文展示，不会随用户变化。

**法律依据：** 用户同意 — 仅在用户主动开启时收集，默认关闭。用户可在「设置 → 隐私 → 云端分析」中开启。

> **说明：** 匿名设备 ID 使用 SHA-256(时间戳 + 进程 ID) 一次性生成并本地持久化，仅用于分析仪表盘中的会话分组。无法通过该 ID 逆向识别您的硬件或身份。

### Level 1 — 聚合统计（可选，Opt-in）

启用后，将收集以下聚合数据：

- 插件启停事件（插件 ID、插件名称、运行时长秒数）
- 快捷启动点击事件（项目 ID、项目类型）
- 更新检测事件（来源：自动检测、尝试次数）

所有数据均经过聚合处理，不包含个人标识信息。

**法律依据：** 用户同意 — 仅在用户主动开启时收集，默认关闭。

### Level 2 — 细粒度行为（可选，Opt-in）

启用后，将收集以下行为数据：

- Tab 切换（路由路径变化）
- 页面停留时长（每个路由的停留秒数）
- 侧边栏切换事件（可见性状态）
- 主题切换事件（暗色/亮色）

需先启用 Level 1 才可开启。

**法律依据：** 用户同意 — 仅在用户主动开启时收集，默认关闭。

## 我们不收集的数据

我们明确**不收集**以下信息：

- 您的姓名、邮箱、电话号码或任何个人身份信息（PII）
- 您系统中的文件内容或文件路径
- 剪贴板内容
- 浏览器历史或凭证
- 硬件序列号

## IP 地址与网络数据

本应用**不会主动采集、传输或存储**您的 IP 地址。发送到分析服务器的任何数据中均不包含 IP 字段。

然而，作为所有 HTTPS 通信的固有特性，自托管的分析服务器（Umami）在网络层会接收到您的来源 IP 地址。这是标准 HTTP 行为，在任何客户端-服务器架构中都无法避免。

**服务器如何使用您的 IP：**

- 聚合地理统计（仅国家级别/地区级别，在仪表盘中显示为「访客国家分布」）
- 机器人检测和垃圾防护（通过 `isbot` 库）
- 会话去重（相同 IP + 相同浏览器 = 同一访客会话）

**您的 IP 如何受到保护：**

- 原始 IP 地址**不会**在分析仪表盘中展示，也不会作为个人标识存储
- IP **不会**与您的匿名设备 ID 以任何可查询的形式关联
- IP **不会**共享给任何第三方服务
- IP **不会**用于个人用户追踪、画像或跨站识别
- 服务端访问日志（如有）仅保留不超过 7 天，仅供安全审计用途

## 数据存储

### 本地存储

本地使用统计（运行时长、打开次数）存储在本地 SQLite 数据库中：

- Windows：`%APPDATA%/my-desktop-tools/stats/stats.db`

此数据永远不会离开您的设备。

### 云端存储

启用云端分析后，数据通过 HTTPS 发送到开发者自托管的 Umami 分析实例。不使用任何第三方分析服务（如 Google Analytics、Mixpanel 等）。

**数据保留：** 云端分析数据保留 90 天，之后自动聚合清理。

## 您的权利

- **访问权：** 您可以查看应用收集了哪些类型的数据（见上方数据收集分级）
- **删除权：** 您可以随时通过「设置 → 隐私 → 删除我的数据」清空已上报的云端分析数据
- **撤回权：** 您可以随时通过「设置 → 隐私」关闭云端分析开关
- **退出权：** 所有可选数据收集均为 opt-in（默认关闭），您无需做任何操作即可保持完全离线

## 安全措施

- 所有网络传输使用 HTTPS 加密
- 分析数据不包含可追溯到您的个人标识
- 匿名设备 ID 为随机生成的 UUID，不基于硬件指纹
- 分析服务器由开发者自建和管理
