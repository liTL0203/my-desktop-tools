下载对应平台的安装包，双击安装即可。

|------|------|
| `latest/My Desktop Tools_0.18.0_x64_en-US.msi` | Windows MSI 安装包 |

## 系统要求

| 平台 | 要求 |
|------|------|
| Windows | Windows 10 及以上（64位） |

## 安装步骤

1. 下载安装包（进入 `latest/` 目录）
2. 双击运行安装程序
3. 安装完成后启动 My Desktop Tools
4. 如需插件：进入 `plugins/` 目录查看各插件说明 → 下载 ZIP → 打开应用 → 插件管理 → 安装插件

## 历史版本

旧版本安装包存放在 `archive/` 目录，按版本号命名：

`archive/v0.3.6/`, `archive/v0.3.5/`, ...
