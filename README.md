## 📋 Installation Steps

1. Click the "MSI Installer" link above to download (`latest/` directory)
2. Double-click to run the installer
3. Launch My Desktop Tools after installation
4. For plugins: download ZIP → open app → **Plugin Manager** → **Install Plugin**

## 📁 Archive Versions

Previous versions are stored in the `archive/` directory, organized by version number:

`archive/v0.3.6/`, `archive/v0.3.5/`, ...
