## 📋 安装步骤

1. 点击上方"MSI 安装包"链接下载安装包（`latest/` 目录）
2. 双击运行安装程序
3. 安装完成后启动 My Desktop Tools
4. 如需插件：下载对应 ZIP → 打开应用 → **插件管理** → **安装插件**

## 📁 历史版本

旧版本安装包存放在 `archive/` 目录，按版本号命名子目录：

`archive/v0.3.6/`, `archive/v0.3.5/`, ...
