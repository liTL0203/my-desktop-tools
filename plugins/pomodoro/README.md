# 番茄钟

> 番茄钟插件 - 25+5 专注模式，自动短休息/长休息切换，统计记录

## 功能介绍

- **专注模式**：25 分钟倒计时，帮助你保持专注
- **短休息**：每完成一个番茄钟后自动进入 5 分钟短休息
- **长休息**：每完成 4 个番茄钟后自动进入 15 分钟长休息
- **自动切换**：专注 → 短休息 → 专注 → ... → 长休息，自动循环
- **统计记录**：记录完成的番茄钟数量，本地持久化存储

## 使用说明

1. 打开 My Desktop Tools
2. 进入左侧导航栏的「番茄钟」页面
3. 点击「开始专注」按钮启动计时
4. 计时结束会自动切换到休息模式
5. 休息结束后继续下一个番茄钟

## 注意事项

- 所有数据存储在本地，不会上传到任何服务器
- 支持亮色/暗色主题，跟随系统设置自动切换
