# Pomodoro Timer

> Focus timer plugin — 25+5 pomodoro technique with automatic short/long breaks and local statistics.

## Features

- **Focus Mode**: 25-minute countdown timer to help you stay focused
- **Short Break**: Automatically enters a 5-minute short break after each pomodoro
- **Long Break**: Automatically enters a 15-minute long break after every 4 pomodoros
- **Auto Switching**: Focus → Short Break → Focus → ... → Long Break, automatic cycle
- **Statistics**: Tracks completed pomodoro count with local persistent storage

## Usage

1. Open My Desktop Tools
2. Navigate to the "Pomodoro" page in the sidebar
3. Click "Start Focus" to begin the timer
4. The timer auto-switches to break mode when finished
5. Continue with the next pomodoro after the break

## Notes

- All data is stored locally and never uploaded to any server
- Supports light/dark theme, following system settings automatically

# 番茄钟

> 番茄钟插件 - 25+5 专注模式，自动短休息/长休息切换，统计记录

## 功能介绍

- **专注模式**：25 分钟倒计时，帮助你保持专注
- **短休息**：每完成一个番茄钟后自动进入 5 分钟短休息
- **长休息**：每完成 4 个番茄钟后自动进入 15 分钟长休息
- **自动切换**：专注 → 短休息 → 专注 → ... → 长休息，自动循环
- **统计记录**：记录完成的番茄钟数量，本地持久化存储

## 使用说明

1. 打开 My Desktop Tools
2. 进入左侧导航栏的「番茄钟」页面
3. 点击「开始专注」按钮启动计时
4. 计时结束会自动切换到休息模式
5. 休息结束后继续下一个番茄钟

## 注意事项

- 所有数据存储在本地，不会上传到任何服务器
- 支持亮色/暗色主题，跟随系统设置自动切换
